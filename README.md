# ☕ ĐƯƠNG Coffee - Mobile E-commerce & Admin Management System

![Project Banner](https://images.unsplash.com/photo-1501339817302-cd4468af3059?ixlib=rb-1.2.1&auto=format&fit=crop&w=1350&q=80)

**ĐƯƠNG Coffee** là một hệ sinh thái ứng dụng mua sắm cà phê hiện đại, bao gồm ứng dụng dành cho khách hàng và hệ thống quản trị (Admin Dashboard) chuyên nghiệp. Dự án được xây dựng với mục tiêu mang lại trải nghiệm mua sắm mượt mà, cao cấp và khả năng quản lý vận hành tối ưu.

---

## ✨ Tính năng nổi bật

### 📱 Dành cho Khách hàng
- **Cửa hàng (Shop):** Giao diện mua sắm trực quan, phân loại danh mục sản phẩm (Cà phê, Trà sữa, Combo...).
- **Giỏ hàng & Thanh toán:** Quy trình đặt hàng nhanh chóng, hỗ trợ nhiều phương thức thanh toán.
- **Lịch sử đơn hàng:** Theo dõi trạng thái đơn hàng thời gian thực.
- **Hệ thống Loyalty:** Tích điểm (Kat Points) và phân hạng thành viên (Silver, Gold, Diamond).
- **Trang cá nhân:** Quản lý thông tin, lưu thông tin đăng nhập tự động.

### 🛠️ Dành cho Quản trị viên (Admin Dashboard)
- **Tổng quan (Stats):** Xem doanh thu, đơn hàng chờ xử lý, tồn kho và người dùng mới qua biểu đồ.
- **Quản lý Nhân sự:** Phân quyền quản trị viên (Super Admin, Admin, Manager), cấp quyền truy cập.
- **Quản lý Kho/Sản phẩm:** CRUD sản phẩm chuyên nghiệp, tải ảnh lên server, quản lý tồn kho và trạng thái bán.
- **Trung tâm Xử lý Đơn hàng:** Tiếp nhận, xác nhận hoặc huỷ đơn hàng thực tế của khách.

---

## 🛠️ Công nghệ sử dụng

- **Frontend:** [Flutter](https://flutter.dev/) (State Management: Provider, UI: Google Fonts, Material Design 3).
- **Backend:** [Node.js](https://nodejs.org/) & [Express](https://expressjs.com/).
- **Database:** [PostgreSQL](https://www.postgresql.org/).
- **Authentication:** JSON Web Token (JWT) & Persistent Login (Shared Preferences).

---

## 🚀 Cài đặt và Chạy dự án

### 1. Backend
```bash
cd backend
npm install
npm run dev
```

### 2. Frontend
```bash
# Đảm bảo đã cài đặt Flutter SDK
flutter pub get
flutter run
```
